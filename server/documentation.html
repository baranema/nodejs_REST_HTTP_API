Server.js documentation - lab group 126

This is REST (Representational state transfer) HTTP API server

Our web service provides our clients with the following functionality:

  1. CRUD (Create, Retrieve, Update and Delete) interface on single rows.
  2. Adds data for a new product item
  3. Lists the data of a specific item
  4. Updates the data of a specific item
  5. Removes the data of a specific item

Server can be started by using the "npm start" command. Installing cors, express and body-parser is also required.

You can run our server on your website with AJAX. For instance, our website index.html is adapted for such use.

End-points

  • http://localhost:3000/ - using GET method displays general information about the server.
  • http://localhost:3000/phones - using GET method displays all products from database.
  • http://localhost:3000/phones/id - using GET method displays a specific item of a given id.
  • http://localhost:3000/phones/brand - using GET method displays a specific items of a given brand.
  • http://localhost:3000/phones - using POST method adds new item to database.
  • http://localhost:3000/phones/id - using PUT method updates a specific item.
  • http://localhost:3000/phones/delete/id - using DELETE method deletes a specifid item.
  • http://localhost:3000/phones/reset - using DELETE method deletes all items.
